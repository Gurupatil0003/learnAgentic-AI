# Understanding Large Language Models (LLMs)
 
## 1. What are LLMs?
Large Language Models (LLMs) are deep learning models trained on large amounts of text data to understand and generate human-like language.

They are widely used for:
- Text generation
- Question answering
- Summarization
- Code generation

---

## 2. Tokens and Tokenization

### Tokens
Tokens are the basic units of text that a model processes.

A token can be:
- A word
- Part of a word
- A character
- A symbol

Example:
"ChatGPT is powerful" → ["Chat", "GPT", " is", " powerful"]

---


### Tokenization
Tokenization is the process of converting text into tokens.

Why it is needed:
- Models do not understand raw text
- They process numerical representations of tokens

## 3. Next-Token Prediction
LLMs work by predicting the **next token** in a sequence.

Given input text, the model:
1. Looks at previous tokens
2. Predicts the most likely next token
3. Repeats the process to generate full output

Example:
Input: "The sky is"  
Output: "blue"

---

## 4. Context Window
The context window is the **maximum number of tokens** a model can consider at once.

Key points:
- Includes both input (prompt) and output
- Limited size (e.g., 4K, 8K, 32K tokens, etc.)
- Older tokens may be ignored if limit is exceeded

Importance:
- Affects memory and coherence
- Larger context → better understanding of long inputs

---



## 5. Prompts and Completions

### Prompt
The input given to the model.

Examples:
- Questions
- Instructions
- Partial text

---

### Completion
The output generated by the model based on the prompt.

Example:
Prompt: "Write a short story about AI"  
Completion: Generated story

## 6. Standard Models vs Reasoning Models

### Standard Models
- Focus on fast and general responses
- Good for:
  - Chat
  - Content generation
  - Basic Q&A

Limitations:
- May struggle with complex reasoning tasks

---

### Reasoning Models
- Designed for step-by-step thinking
- Better at:
  - Logical reasoning
  - Problem solving
  - Multi-step tasks

Characteristics:
- Slower but more accurate for complex problems
- Often use structured reasoning internally

---

### Comparison

| Feature | Standard Models | Reasoning Models |
|--------|----------------|------------------|
| Speed | Fast | Slower |
| Accuracy (simple tasks) | High | High |
| Accuracy (complex tasks) | Moderate | High |
| Use Case | General tasks | Logical reasoning, problem solving |

---

## 7. Summary
LLMs process text using tokens and generate output through next-token prediction.

Their effectiveness depends on:
- Tokenization
- Context window size
- Quality of prompts

Different types of models are optimized for either speed (standard) or deeper reasoning (reasoning models).
